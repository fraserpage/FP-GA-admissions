A note about this site

One of my coding goals is always to avoid repeating myself. DRY. In that spirit, because I didn't want to have to write out the same HTML markup over and over again for each of the 'accordion' elements above, and to give myself the flexibility to easily change that markup if I wanted to, I've written that section of the site inside of a Javascript object array and then loop through that array to create the accordion. In the real world I'd use a CMS for this to give myself the niceties of a real text editor. I'm sure this page is riddled with typos as a result of my doing my writing in a code editor. Sorry!

Note that I'm loading in my js from my github here. I'm not sure exactly what was happening but GA's Dash platform was erasing '+'s and doing other funny things to inline javascript code when loading it up on the 'fullscreen' url.
